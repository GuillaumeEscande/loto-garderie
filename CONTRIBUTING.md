# Contribuer

Merci de votre intérêt pour le projet.

## Signaler un bug ou proposer une idée

Ouvrez une [issue](https://github.com/VOTRE_USER/garderie/issues) en décrivant le problème ou la suggestion.

## Proposer une modification (Pull Request)

1. Fork du dépôt, puis cloner et créer une branche.
2. Faire vos changements (PHP, HTML/CSS, pas de dépendances lourdes si possible).
3. Tester avec `docker compose up --build` et vérifier que l’app fonctionne.
4. Ouvrir une Pull Request en expliquant brièvement le but des changements.

On garde le projet simple : MVC (src/Controller, src/Model, src/View), SQLite, formulaire/POST classiques, Tailwind en CDN.
